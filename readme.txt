Adds WOT rating symbols to your posts to indicate which links are safe

== Description ==

WOT for Blogs is a <a href="http://www.wordpress.org" class="norating">Wordpress</a> plugin which adds WOT rating symbols to your blog and blog comments. Keep your users safe and warn them of any potentially malicious/untrustworthy links.

Note: Only outgoing links in posts/comments have the WOT rating symbols added to them

== Installation ==

1. Upload the 'wotforblogs' directory to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I adjust where rating symbols appear? =

Included with the plugin is a wotforblogs-config.js file. This file contains configuration and a list of CSS selectors where the rating symbols will be applied. You can add to this list to get rating symbols to appear in different plces.
